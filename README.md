# AI Competitiveness Assessment System

A comprehensive six-dimensional analysis and prediction system for assessing AI competitiveness across 10 major countries.

## Features

- **Six-Dimensional Analysis**: Talent Reserve, Data Resources, Industrial Ecosystem, Infrastructure, Technological Innovation, Policy Support
- **10 Major Countries**: USA, China, UK, Germany, South Korea, Japan, France, Canada, UAE, India
- **Time Period**: Historical (2020-2025) and Future Predictions (2026-2035)
- **Advanced Analytics**: Entropy-based weight calculation, multi-dimensional scoring
- **Comprehensive Visualizations**: 9-panel dashboard, heatmaps, radar charts, trend analysis
- **Strategic Insights**: Country rankings, growth trajectories, recommendations
